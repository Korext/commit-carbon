# Originality Statement

Every component of Commit Carbon is original work by Korext.

## Methodology

The calculation methodology in METHODOLOGY.md is independently developed. It uses principles from GHG Protocol, academic research, and established emissions accounting practices but does not copy text or formulas from any source.

## Emissions Factors

All emissions factors in factors.yaml are independently derived from publicly available data:

- Inference energy estimates are based on academic research (primarily Luccioni et al. 2023 and Patterson et al. 2021) and adapted for AI coding tool usage patterns.
- Grid carbon intensity data is sourced from public datasets (IEA, EPA eGRID) with proper attribution.
- No vendor proprietary data is used.

## Report Formats

Disclosure report formats (CSRD, SEC, CDP, GHG Protocol) are structured to be compatible with each framework's requirements but do not copy framework text, templates, or schemas.

## Code

All code (CLI, GitHub Action, web dashboard, API) is original. The sole dependency is js-yaml for YAML parsing.

## What This Project Does Not Contain

- No copied text from GHG Protocol, CDP, SBTi, or ISO standards
- No copied regulatory text from CSRD, SEC rules, or California statutes
- No paraphrased academic papers
- No vendor proprietary emissions data
- No Korext platform infrastructure code
